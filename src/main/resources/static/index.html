SERVICE

Aplication returns image according to a search term.
The image can be returned from 2 places: Database / pixabay API

- /getAllImages - returns list of all images in database

- /getImage/{term} - returns image by term. If term not exist in DB. Adding to DB value that received from pixabay

Url example:
https://pixabay.com/api/?key=SOME_KEY&q=yellow+flowers&image_type=photo&pretty=true




For tests:

- /greet (GET, POST) - returns json: {"id": 1, "message": "Hello World!!!"}

- /greet/{name} (GET) - returns json with name: {"id": 1, "message": "Hello World " + name + "!!!"}

- /greetParams (GET) - returns json: {"id": 1, "message": "Hello World " + name + " " + surname + "!!!"}
